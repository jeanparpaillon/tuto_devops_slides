# Day 1 — Core concepts

## Automate from hardware to application

### Objectives

- Understand the DevOps culture and its pillars
- Discover the technical stack: libvirt/VMware, Terraform, Kubernetes, Ansible

### Content

- Introduction to DevOps: culture, CI/CD, IaC
- Production architecture: virtualization, containers, orchestration
- Tools overview:
  - VMware vSphere: virtual infrastructure management
  - Terraform: infrastructure as code
  - Kubernetes: container orchestration
  - Ansible: deployment and configuration
- Environment setup: tools, Git repo, SSH

# Training objectives

- Understand the foundations of DevOps in production
- Deploy and orchestrate a complete infrastructure VMware → Kubernetes
- Automate the deployment of internal and external applications with Ansible
- Explore best practices for security, monitoring, and high availability
